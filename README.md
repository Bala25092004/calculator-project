 Calculator Project

 Project Overview

This project is a fully functional Calculator built using HTML, CSS, Bootstrap, and JavaScript.
It is developed after learning the basics of frontend technologies to understand core concepts and implement them in a real-world usable application.

The calculator supports both whole numbers and decimal calculations and includes all essential arithmetic operations.

⦁	 Features

⦁	 Addition

⦁	 Subtraction

⦁	 Multiplication

⦁	 Division

⦁	 Modulo operation

⦁	 Square of a number

⦁	 AC (All Clear) button to reset input

⦁	 Equal (=) button to calculate results

⦁	 Supports decimal and whole numbers

⦁	 Button click handling using JavaScript event listeners

Technologies Used

⦁	HTML – Structure of the calculator

⦁	CSS – Styling and layout

⦁	Bootstrap – Responsive design and UI components

⦁	CSS Flexbox / Grid – Layout arrangement

⦁	JavaScript –
      
                Event Listeners

                Multiple Functions

                Loops

                if/else Conditions

                innerText / innerHTML

 No external JavaScript libraries used
 Own logic implemented (no copy-paste from Google)

 How It Works

⦁	User clicks buttons on the calculator

⦁	JavaScript captures input using event listeners

⦁	Operations are stored and evaluated using custom logic

⦁	Result is displayed dynamically using innerText

⦁	AC button clears all data

⦁	Equal button performs calculation
